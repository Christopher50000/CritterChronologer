# Critter Chronologer

## Overview

Critter Chronologer is a pet scheduling application developed as part of the Udacity Java Web Developer Nanodegree program. The application enables pet owners to schedule services for their pets while allowing employees to manage appointments and service availability.

## Features

* Manage customers and their pets
* Schedule pet service appointments
* Track employee availability and skills
* Assign employees to service requests
* RESTful API endpoints for scheduling operations
* Persistent data storage using a relational database

## Technologies Used

* Java
* Spring Boot
* Spring Data JPA
* Hibernate
* Maven
* H2/MySQL Database
* REST APIs

## Architecture

The application follows a layered architecture:

* Controllers handle API requests
* Services contain business logic
* Repositories manage data persistence
* Entities model customers, pets, employees, and schedules

## Learning Objectives

This project demonstrates:

* Spring Boot application development
* REST API design
* Database persistence with JPA
* Object-relational mapping (ORM)
* Service-layer architecture
* Backend application development
